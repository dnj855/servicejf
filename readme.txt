(version 1)

Le service JF est une invention de deux joyeux lurons salariés d'une radio
de service public en Lorraine. L'idée de ce site est de créer un portail unique
pour tous les membres du service, c'est-à-dire plus ou moins tous les salariés
de cette radio.

A ce jour, quatre jeux existent :
- le challenge invité (abrégé ci)
- le challenge des soirées sport (abrégé css)
- le challenge du fichier gnou (abrégé fg)
- le challenge de la présidentielle (abrégé cp)

Le tout est développé entièrement à la main et est parfaitement open source.

A également été créé, par le passé, le challenge phénoménal handball (cph).

Parmi les angles d'amélioration prévus, outre la création de nouveaux
challenges, je souhaite transformer le site pour le faire fonctionner avec
symfony. D'ici peu, je mettrai en ligne le git de la version "symfonifiée" en
cours de développement.
Autre amélioration permanente, le design responsive grace à Bootstrap.

If you want, jump in !

Cédric Lang-Roth, développeur-programmeur
